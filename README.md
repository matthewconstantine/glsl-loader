# GLSL loader for webpack

## Installation

`npm install glsl-loader`

## Usage

``` javascript
var fileContent = require("glsl!./shader.glsl");
// => returns shader.glsl content as string
```

[Documentation: Using loaders](http://webpack.github.io/docs/using-loaders.html)
